# 📊 Placement Tracker

A simple web-based Placement Tracker application built using Python (Flask), HTML, CSS, and SQLite.  
It helps students track coding problems, progress, and placement preparation in a structured way.

---

## 🚀 Features

- Add and manage coding problems
- Track solved / unsolved problems
- Store data locally using SQLite database
- Simple and clean UI using HTML & CSS
- Beginner-friendly Flask backend

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML
- CSS
- SQLite
- JavaScript
